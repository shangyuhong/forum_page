遊戲介紹

激鬥峽谷是什麼？

縱身一躍進入《激鬥峽谷》的世界：為手機和家用主機從零開始重新打造，完美再現《英雄聯盟》中講求技巧與戰術的五對五MOBA體驗。遊戲中有全新的控制方式和快節奏的對戰，新手與高手都能與好友組成隊伍、鎖定自己的拿手英雄，然後大顯身手。

遊戲目標

一、選擇你的路線

將你最棒的射手或輔助放在雙人線，讓法師在中路使出魔法連發，或者用鬥士牢牢守住單人線。

二、游擊打野

除了專注對線，你也可以進入野區殺怪。討伐巴龍納什或飛龍等史詩級野怪，來幫隊伍獲獲得強大的增益效果。

三、摧毀防禦塔來進攻主堡

通往敵方基地的每條路線都有防禦塔在把守。摧毀這些建築，鋪出迎接勝利的道路。

四、摧毀敵方基地

雙方隊伍基地的中心就是主堡。保護己方主堡，先一步推毀敵方的主堡來赢得對戰。
